# Safe Steps – A Web-Based Support and Awareness Platform for Parents of Children with Special Needs

Safe Steps is a socially relevant web platform designed to support parents of children with developmental and sensory disabilities. The platform provides early awareness, structured guidance, community interaction, and verified local support resources for disabilities such as blindness, deafness, dyslexia, and speech impairment.

The system integrates a rule-based assessment engine, multilingual accessibility (English & Malayalam), district-wise resource filtering, and a parent community forum within a secure web environment.

---

# 🌟 Features

* 📚 Learn module with disability information and early signs
* 🧠 Rule-based disability assessment questionnaire
* 🏥 Resource finder with district-wise and disability-wise filtering
* 👥 Community forum for parent interaction and support
* 🌐 Multilingual support (English & Malayalam)
* 🔒 Secure authentication and session management
* 🔗 REST API based frontend-backend communication
* 📊 High performance and accessibility evaluation using Google Lighthouse

## 🧠 Assessment Module

A questionnaire-based early screening system that:

* Collects symptom responses from parents
* Uses rule-based scoring logic
* Categorizes risk levels
* Provides preliminary guidance

⚠️ *Note: This is not a medical diagnostic tool.*

---

# 🔄 Workflow

1. User interacts with the frontend interface
2. Frontend sends requests to backend using REST APIs
3. Backend processes:

   * Authentication
   * Assessment scoring
   * Forum operations
   * Resource filtering
4. MySQL database stores and retrieves data
5. Results are returned to the frontend and displayed to users

## 📌 Conclusion

Safe Steps bridges the gap between early symptom recognition and professional consultation by combining awareness resources, early screening guidance, verified support services, and community-based emotional support into a single digital platform.

The platform aims to create a safe, accessible, and supportive environment for parents of children with special needs.
